What is inside a hard drive?

 

Inside a modern hard drive

  1. The actuator that moves the read - write arm around 
  2. The read - write arm which moves the read - write head around across the platter 
  3. The Central spindal which moves the patters
  4. The platter itself which stores information in binary code (1s and 0s)
  5. The plug which connects the hard drive to the rest of the computer
  6. A tiny magnet that reads data called the read - write head
  7. Circuit which controls data going from and to the platter
  8. Flexible wire connecting the circuit to the read - write head
  9. Spindle allowing the read - write arm to navigate acrowss the platter
