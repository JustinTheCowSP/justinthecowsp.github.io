If you type this up (...or preferably copy and paste it :P) into a HTML document and preview in your browser, you will have falling snow! Now, our implementation isn't obviously as simple as what you see here. A lot of the heavy lifting is done in the falling snow JavaScript file (fallingsnow_v6.js) that this document references, so let's take a quick look at that: // The star of every good animation var requestAnimationFrame = window.requestAnimationFrame || window.mozRequestAnimationFrame || window.webkitRequestAnimationFrame || window.msRequestAnimationFrame;
